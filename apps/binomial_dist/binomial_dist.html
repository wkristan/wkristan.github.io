The binomial formula, for your reference, is:

P ( X ) = N ! X ! ( N − X ) ! p X ( 1 − p ) N − X P( X ) = { N! } over { X!(N-X)! } p^X( 1-p )^{ N-X }

Dividing it into its probability part and its counting part makes it easier to see how it's working. Change any of the three things you can, number of eggs, probability of a female, or number of females in a clutch, and see how the calculation is done.

Probability part = p X ( 1 − p ) ( N − X ) = 0.5 0 ( 1 − 0.5 ) ( 2 − 0 ) = 0.25 "Probability of any single way to get X successes out of N trials" = p^X (1-p)^( N-X ) = 0.5^27 = 7.45 times 10^{

The "counting part" of the formula gives the number of different ways that 0 females out of 2 could happen:

Counting part = N ! X ! ( N − X ) ! = 2 ! 0 ! ( 2 − 0 ) ! = 1 "Number of ways to get X successes out of N trials" = { N! } over { X!(N-X)! } = { 27! } over { 18!(27-18)! } = 4686825

The probability part and counting part are multiplied together to get the binomial probability:

P ( 0 ) = counting × probability = 1 × 0.25 = 0.5
