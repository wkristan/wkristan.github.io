ANOVA and the F Distribution for Three Groups

This version is also a tutorial. It still runs the simulation, but the page and source code now explain how the HTML, CSS, and JavaScript fit together.

How this page works

1. HTML structure

The HTML creates named elements like samplePlot, fPlot, and anovaTable. JavaScript later finds those elements and fills them with graphs or text.

2. CSS styling

CSS controls layout and appearance. Classes like .panel, .grid, and .controls make the page look like a polished dashboard without changing the underlying data.

3. JavaScript behavior

JavaScript creates random samples, computes ANOVA quantities, updates the table, and tells Plotly what to draw. Button clicks trigger all of that behavior.

4. Plotly graphs

Plotly turns arrays of numbers into interactive charts. Each graph is built from one or more “traces,” such as dots, lines, markers, or histograms.

Random samples and group means

The dots are raw data values for the three groups. Horizontal line segments mark each sample mean. ANOVA compares variability between group means to variability within groups.

Why the buttons work

Each button has an id. In JavaScript, we use document.getElementById(...).addEventListener("click", ...) to attach a function to each button.

Why inputs matter

The input boxes store user choices. JavaScript reads them using parseInt or parseFloat right before generating the samples.

Sampling distribution of F statistics

The histogram shows the empirical distribution of F statistics from the samples collected so far. The theoretical F curve is included in the legend as F distribution curve and starts hidden; click its legend entry to overlay it.

Reading the JavaScript

State variables

The array fValues stores every simulated F statistic. The variable latestAnova stores the most recent ANOVA result so the newest F value can be highlighted on the histogram.

Small helper functions

Functions like randn, sampleNormal, mean, and fmt each do one simple job. This makes the code easier to test and easier to explain.

ANOVA calculation

anovaOneWay computes sums of squares, degrees of freedom, mean squares, and the F ratio. That function is the statistical core of the page.

Rendering

Functions beginning with update... do not calculate new data. They take existing results and render them into the page or into Plotly graphs.
