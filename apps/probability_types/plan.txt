    If marginal
        Clear any joint, given, conditional
        Set text color, background color for the cells used
        Return row total/grand total.

    If joint
        Clear any marginal, conditional, given
        Check if another joint is already selected
            Yes - set text, background color
                Return cell count/grand total

                When second choice is made, set text color, background color for cells used
                Return cell count/grand total

    If given
        Clear any marginal, given, joint
        Check if a conditional is selected
            Yes - set text, background color
                Return cell count / marginal total
            No - Prompt for a conditional choice
                When marginal is made, set text color, background color
                Return cell count / marginal total

    If conditional
        Clear any marginal, conditional, joint
        Check if a given is selected
            Yes - set text, background color
                Return cell count / marginal total
            No - prompt for a given
                When given is selected, set text color, background color
                Return cell count / marginal total
